# Deep Learning for Audio Deepfake Detection

**Evaluation of State-of-the-Art Models and Robustness Analysis on Custom Datasets**

Bachelor's project — Antton Correia Dupuy
Multimedia Signal Processing Group (MMSPG), EPFL · Spring 2026
Supervisors: Robin Szymczak, Enes Eray Demirtas · Professor: Touradj Ebrahimi

## Overview

Modern text-to-speech (TTS) and voice-conversion systems can now synthesize speech that is nearly indistinguishable from genuine recordings, fueling fraud and disinformation. This project evaluates four state-of-the-art **open-source** audio deepfake detectors and asks a single question: *does strong benchmark performance translate into real-world robustness?*

Each model is first validated on the public **In-the-Wild (ITW)** benchmark, then stress-tested on three custom datasets of increasing difficulty, whose spoofed samples are generated by **top-ranked TTS systems from the TTS Arena 2 leaderboard**.

**Two key findings:**

1. Spoofed audio from modern TTS systematically evades detectors trained on older corpora.
2. Even the best detectors fail on specific categories of *genuine* audio — deep/gravelly voices, non-lexical vocalizations (e.g. onomatopoeia), and fast or heavily accented speech. On a dataset built to isolate this effect, DF Arena 1B and 500M drop to near-random performance (48% and 37% EER).

The takeaway: a strong score on a fixed benchmark does not guarantee robustness, and **vocal register and phonetic diversity are a real, underexplored axis of vulnerability**.

## Models evaluated

| Model | Front-end | Back-end | Params | Training data |
|---|---|---|---|---|
| XLS-R + SLS | XLS-R (SSL) | SLS | ~340M | ASVspoof 2019 |
| Wav2Vec2-AASIST (CodecFake) | wav2vec2 (SSL) | AASIST | ~316M | CodecFake (2024) |
| DF Arena 500M | SSL backbone | n/d | ~500M | Arena multi-dataset recipe |
| DF Arena 1B | SSL backbone | n/d | ~1B | Arena multi-dataset recipe |

All detectors share a two-stage front-end / back-end structure and output a continuous score; performance is measured with the **Equal Error Rate (EER)**.

## Datasets

Three custom evaluation sets were built; each pairs bona fide recordings with spoof samples from current TTS systems. The folder name encodes the bona fide / spoof split.

| Folder | Set | Size (bona fide / spoof) | Focus |
|---|---|---|---|
| `DataSet_50_50/` | Set I — Balanced English | 50 / 50 | Synthesis quality at the current frontier in a controlled English setting. Bona fide from LibriSpeech, LJ Speech and CSTR; spoof from CastleFlow v1.0, Inworld TTS MAX, Kokoro 82M, CosyVoice 2, StyleTTS 2. |
| `DataSet_10_10/` | Set II — Multilingual personal | 10 / 10 | Multilingual content (EN/FR/ES/IT/PT), voice cloning, and real-world acoustic degradation (festival noise, VoIP, loudspeaker capture). |
| `Dataset_30_30/` | Set III — Adversarial | 30 / 30 | Isolates *vocal register* as the variable: the same sentence recorded in neutral / deep / fast registers, plus non-lexical vocalizations. Designed to break the detectors. |

Each dataset folder contains `bonafide/`, `spoof/`, and `images/` (per-model score plots and confusion matrices).

## Results

**In-the-Wild — published vs. reproduced EER (%)**

| Model | Published | Reproduced |
|---|---|---|
| XLS-R + SLS | 7.46 | 7.46 |
| Wav2Vec2-AASIST (CodecFake) | 9.6 | 4.69 |
| DF Arena 500M | 1.86 | 1.76 |
| DF Arena 1B | 0.9 | 0.9 |

**Custom evaluation sets — EER (%) / Accuracy (%)**

| Model | Set I (50/50) | Set II (10/10) | Set III (30/30) |
|---|---|---|---|
| XLS-R + SLS | 36.0 / 64.0 | 50.0 / 40.0 | — |
| Wav2Vec2-AASIST | 24.0 / 76.0 | 45.0 / 55.0 | — |
| DF Arena 500M | 2.0 / 97.0 | 15.0 / 80.0 | 37.0 / 65.0 |
| DF Arena 1B | 4.0 / 96.0 | 5.0 / 95.0 | 48.0 / 52.0 |

The two older models (XLS-R + SLS, Wav2Vec2-AASIST) collapse on modern TTS and were excluded from Set III; that analysis focuses on the two stronger DF Arena models — which still drop to near-random performance on the targeted slice.

> **Note:** the custom sets are small (n = 20–60), so absolute error rates carry wide confidence intervals and should be read as evidence of near-random behaviour rather than precise measurements.
